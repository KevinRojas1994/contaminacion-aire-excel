# Análisis de Contaminación del Aire – Limpieza de Datos en Excel

Este proyecto fue desarrollado como un ejercicio práctico de análisis de datos,
enfocado en la limpieza, organización y preparación de información sobre calidad
del aire para su posterior análisis.

## Objetivo del proyecto
Limpiar y preparar un dataset de contaminación del aire para garantizar la calidad
de los datos y facilitar su análisis, identificando errores, valores faltantes e
inconsistencias.

## Descripción del dataset
El dataset contiene información simulada sobre contaminación del aire en diferentes
ciudades, incluyendo variables como:
- Ciudad
- Fecha
- Contaminante (PM2.5, PM10, NO2, O3)
- Nivel de concentración
- Unidad de medida

## Herramientas utilizadas
- Excel
- GitHub

## Proceso de limpieza de datos
1. Eliminación de registros duplicados.
2. Corrección de errores en nombres de ciudades y contaminantes.
3. Tratamiento de valores nulos y faltantes.
4. Estandarización de formatos de fecha y unidades.
5. Validación de rangos de concentración.

## Resultados
Luego del proceso de limpieza, el dataset quedó listo para su análisis,
permitiendo identificar patrones y niveles de contaminación de forma confiable.
